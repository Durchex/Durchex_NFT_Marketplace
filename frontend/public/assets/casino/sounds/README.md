# Casino sounds

Place audio files here (MP3 or OGG). Filenames used by the app:

- `dice-roll.mp3` – dice rolling
- `dice-land.mp3` – dice stop
- `card-flip.mp3` – card flip/deal
- `slot-spin.mp3` – slot reel spin
- `slot-win.mp3` – slot win
- `roulette-spin.mp3` – roulette wheel
- `roulette-ball.mp3` – ball landing
- `chip-place.mp3` – chip/bet place
- `win.mp3` – generic win
- `lose.mp3` – generic lose
- `button-click.mp3` – UI click

- [Freesound.org](https://freesound.org) – search "casino", "dice", "slot"
- [Pixabay Sound Effects](https://pixabay.com/sound-effects/search/casino/)
